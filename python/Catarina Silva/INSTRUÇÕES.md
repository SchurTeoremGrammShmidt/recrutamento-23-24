# Como executar: #
Escrever "python3 task_manager.py" no terminal.

# Como funciona: #
Este programa tem como objetivo fazer o armazenamento e a gestão de tasks. Nele é possível ter acesso às seguintes funcionalidades:

- Criar uma nova task (tecla 1)
- Editar o nome e/ou descrição de uma task já existente (tecla 2)
- Apagar uma task (tecla 3)
- Ver as tasks guardadas (tecla 4)
- Sair do programa (tecla 0)

Dentro de cada uma destas funcionalidades existem mais opções, contudo estão sempre numeradas para que seja simples a interação com o utilizador. Adicionalmente, as tasks são armazenadas no ficheiro "tasks.txt", que é criado na pasta onde está o executável do programa, sendo as tasks guardadas mesmo que o programa seja encerrado.
